# Memory Card Game Project


A fun and interactive Memory Card Game that allows users to:

Flip cards to find matching pairs

Track their score

Restart the game anytime

Improve memory and focus through gameplay


## 📝 What Does It Do?

The app allows users to play a simple memory game with a grid of cards:

Flip Cards: Users click on a card to reveal its hidden image.

Match Pairs: Find the matching card to earn points.

Score Tracking: Shows the current score based on matched pairs.

Restart Game: Resets the game board for a new round.

The interface is clean and minimal, focusing on usability and a smooth gaming experience.


## ⚙️ How Does It Work?

Single-page structure: Built with HTML, CSS, and JavaScript.

Event-driven logic: Clicking a card triggers the flip and match-check logic.

Score management: Increases the score when a matching pair is found.

Dynamic DOM updates: Cards flip and disappear or remain based on match results, all without refreshing the page.

Lightweight implementation: No external libraries — pure JavaScript and DOM manipulation.
